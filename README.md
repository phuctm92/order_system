# Readme
App name: Order System

## 1. Prerequisite
-   Ruby - 2.7.1
-   Rails - 6.1.3.1
-   Yarn
-   Webpack
-   jQuery
-   Bootstrap

## 3. Installation
```
git clone https://github.com/phuctm92/order_system-.git
cd order_system
bundle install
yarn add bootstrap@4.3.1 jquery popper.js
rails db:create && rails db:migrate && rails data:migrate
rails s
```
